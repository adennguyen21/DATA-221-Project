# DATA-221-Project
## Heart Failure Prediction using Machine Learning
## Authors

Aden Nguyen, Jacob Jeon, Uinsean Pamplona, Patrick Macarilay
- Department of Computer Science, University of Calgary

## Project Overview

This project applies machine learning models to predict the presence of heart disease using clinical patient data. The task is treated as a binary classification problem where the goal is to determine whether a patient has heart disease (1) or not (0).

## Note: The LinearRegression.ipynb is the old version of our logistic regression model, the current version is in Logistic Regression Model.py.
